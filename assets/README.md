# Application Icons

Place your application icons here:

- **icon.png** (256x256) - Application icon for desktop shortcuts
- **icon.ico** - Windows icon file
- **icon.icns** - macOS icon file (if building for Mac)

Recommended: Create a professional logo and save as PNG, then convert using:
- Online: https://icoconvert.com
- Or use ImageMagick: `convert icon.png icon.ico`
